## 修复 GitHub Actions 构建错误

### 问题根源
代码中使用了 ViewBinding 类（`ActivityMainBinding` 和 `ActivityOrganizeResultBinding`），但缺少必要的导入语句。由于移除了手动实现的类，现在需要使用自动生成的 ViewBinding 类。

### 修复步骤

1. **在 MainActivity.kt 中添加 ViewBinding 导入**
   - 添加：`import com.apporganizer.databinding.ActivityMainBinding`
   - 位置：在现有导入语句之后

2. **在 OrganizeResultActivity.kt 中添加 ViewBinding 导入**
   - 添加：`import com.apporganizer.databinding.ActivityOrganizeResultBinding`
   - 位置：在现有导入语句之后

### 预期结果
- Kotlin 编译器能够找到自动生成的 ViewBinding 类
- GitHub Actions 构建成功
- APK 正常生成
