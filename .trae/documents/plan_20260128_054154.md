## 修复 Kotlin 编译错误

### 问题根源
`FolderInfo` 类在 `OrganizeResultActivity.kt` 中定义，但被 `AppOrganizerAccessibilityService.kt` 使用。由于 `FolderInfo` 不是公共的独立类，`AppOrganizerAccessibilityService` 无法访问它，导致编译失败。

### 修复步骤

1. **将 FolderInfo 类移动到 AppInfo.kt**
   - 从 `OrganizeResultActivity.kt` 中移除 `FolderInfo` 类定义
   - 将 `FolderInfo` 类添加到 `AppInfo.kt` 文件中
   - 这样所有类都可以访问 `FolderInfo`

2. **更新 OrganizeResultActivity.kt**
   - 移除 `FolderInfo` 类定义（第 244-247 行）

### 预期结果
- `AppOrganizerAccessibilityService` 可以正常访问 `FolderInfo`
- Kotlin 编译成功
- GitHub Actions 构建成功
