# 📱 安装指南 - 应用整理工具

## 🎯 目标
将这个应用安装到你的 Android 手机上，让它读取并整理你手机上的所有应用。

---

## 🚀 快速开始（推荐新手）

### 准备工作
1. **一台 Windows 电脑**（你现在用的）
2. **一部 Android 手机**
3. **一根数据线**
4. **20分钟时间**

---

## 📋 详细步骤

### 第一步：安装 Android Studio

1. **下载 Android Studio**
   - 访问：https://developer.android.com/studio
   - 点击"Download Android Studio"
   - 下载大约 1GB，耐心等待

2. **安装 Android Studio**
   - 双击下载的 exe 文件
   - 一路"Next"，保持默认设置
   - 首次启动会下载额外组件（约10-15分钟）

### 第二步：打开项目

1. **启动 Android Studio**
2. **点击"Open"**
3. **选择文件夹**：
   ```
   C:\Users\EDY\CodeBuddy\20260126090212\android
   ```
4. **等待项目加载**（首次需要下载依赖，约5-10分钟）

### 第三步：配置手机

1. **开启开发者模式**
   - 打开手机"设置"
   - 找到"关于手机"
   - 连续点击"版本号"7次
   - 提示"您已处于开发者模式"

2. **开启 USB 调试**
   - 返回"设置"
   - 找到"开发者选项"（可能在"系统"或"更多设置"里）
   - 开启"USB 调试"
   - 如果有"USB 安装"也开启

3. **连接手机**
   - 用数据线连接手机和电脑
   - 手机会弹出"允许USB调试"提示
   - 勾选"始终允许"，点击"确定"

### 第四步：运行应用

1. **在 Android Studio 中**
   - 顶部工具栏会显示你的手机型号
   - 点击绿色的 ▶️ 运行按钮
   - 等待编译和安装（首次约2-3分钟）

2. **手机上**
   - 应用会自动安装并打开
   - 首次启动会加载所有应用（约5-10秒）

3. **完成！**
   - 你会看到手机上所有应用被自动分类
   - 可以切换品牌风格、搜索、筛选分类

## 🔧 常见问题

### Q1: Android Studio 下载太慢？
**A**: 使用国内镜像：
- 访问：https://developer.android.google.cn/studio
- 或使用迅雷等下载工具

### Q2: 手机连接不上？
**A**: 检查以下几点：
1. 数据线是否支持数据传输（不是只充电的线）
2. USB调试是否开启
3. 手机是否信任这台电脑
4. 尝试换个USB口

### Q3: 找不到"开发者选项"？
**A**: 不同品牌位置不同：
- **小米**：设置 → 更多设置 → 开发者选项
- **华为**：设置 → 系统和更新 → 开发者选项
- **OPPO**：设置 → 关于手机 → 版本号（点7次）→ 返回 → 其他设置 → 开发者选项
- **vivo**：设置 → 更多设置 → 开发者选项

### Q4: 编译报错？
**A**: 常见解决方案：
1. 等待所有下载完成（查看底部状态栏）
2. 点击 Tools → SDK Manager，确保安装了 Android SDK 34
3. File → Invalidate Caches → Invalidate and Restart

### Q5: 应用安装失败？
**A**: 
1. 手机开启"USB安装"或"通过USB安装应用"
2. 关闭手机的"纯净模式"（如果有）
3. 给予安装权限

### Q6: 只看到很少的应用？
**A**: 这是 Android 11+ 的限制：
- 开发版本会过滤部分系统应用
- 正式上架应用商店后可获取完整列表
- 常用的微信、QQ、淘宝等都能正常显示
