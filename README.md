# 📱 应用整理工具

一个 Android 原生应用，可以**自动读取并智能整理手机上的所有应用**。

## ✨ 核心功能

- ✅ **智能分析**：自动读取手机上所有应用并分类
- ✅ **整理方案**：根据偏好生成文件夹整理建议（常规/工作/娱乐/极简）
- ✅ **15种分类**：社交、办公、娱乐、音乐、购物、金融、旅游、健康、新闻、照片、教育、工具、游戏等
- ✅ **品牌适配**：自动检测手机品牌（小米/华为/OPPO/vivo）
- ✅ **可视化展示**：清晰显示每个文件夹应该放哪些应用
- ✅ **操作指导**：详细说明如何手动创建文件夹整理

## 🚀 快速开始

### 📥 下载安装（推荐）

1. 访问 [Releases 页面](../../releases)
2. 下载最新的 `App-Organizer-vX.X.apk`
3. 传到手机安装
4. 打开应用，选择整理偏好，点击"开始整理"

### ⚡ 本地快速测试（开发者）

**推荐开发流程**：本地构建测试 → 确认无误 → 提交GitHub

```batch
# 方式1: 本地构建（手动传输到手机）
build_local.bat

# 方式2: 构建并通过USB自动安装（需要ADB）
build_and_install.bat

# 方式3: 快速测试菜单
quick_test.bat
```

**详细指南**：[本地构建文档](LOCAL_BUILD_GUIDE.md)

**优势**：
- ⚡ 1-2分钟快速构建
- 🔄 支持快速迭代测试
- 📱 可通过ADB直接安装
- ✅ 测试无误后再提交GitHub

## 📋 智能分类

应用会根据**包名**和**应用名**自动分类（支持中英文）：

- **💬 社交通讯**：微信、QQ、微博、抖音、小红书等
- **💼 办公学习**：钉钉、企业微信、WPS、飞书、印象笔记等
- **🎬 影音娱乐**：爱奇艺、腾讯视频、优酷、哔哩哔哩等
- **🎵 音乐视频**：网易云音乐、QQ音乐、酷狗、喜马拉雅等
- **🛒 购物支付**：淘宝、京东、拼多多、美团、饿了么等
- **💰 金融理财**：支付宝、各大银行App、证券App等
- **✈️ 出行旅游**：高德地图、百度地图、滴滴、携程等
- **💊 运动健康**：Keep、运动类、健康医疗类App
- **📰 新闻资讯**：今日头条、新闻类、阅读类App
- **📷 拍照摄影**：相机、相册、美图、图片类App
- **📚 教育学习**：学习类、课程类、作业类App
- **🔧 实用工具**：百度网盘、浏览器、文件、输入法等
- **🎮 游戏娱乐**：王者荣耀、原神、游戏类App

## 🛠️ 技术栈

## 🔧 开发流程

### 推荐工作流（快速迭代）

```batch
# 1. 修改代码
编辑 MainActivity.kt / 布局文件等

# 2. 本地快速测试（1-2分钟）
build_and_install.bat    # 或 build_local.bat

# 3. 在手机上测试功能

# 4. 确认无误后提交
git add -A
git commit -m "功能描述"
git push

# 5. GitHub Actions 自动构建并发布 Release
```

**对比**：
- 本地测试：**1-2分钟**反馈 ⚡
- GitHub CI：**4-6分钟**反馈 🐌

### 构建命令

```batch
# 清理构建
cd android && gradlew.bat clean

# 构建Debug版
cd android && gradlew.bat assembleDebug

# 通过ADB安装
adb install -r android/app/build/outputs/apk/debug/app-debug.apk
```
