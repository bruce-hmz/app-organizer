# 📱 应用整理工具

一个 Android 原生应用，可以**自动读取并整理手机上的所有应用**。

## ✨ 功能特点

- ✅ **自动读取手机应用**：读取所有已安装的应用
- ✅ **智能分类**：根据包名自动分类（社交/办公/娱乐/工具/购物/金融/游戏）
- ✅ **多分类支持**：一个应用可以属于多个分类
- ✅ **品牌UI适配**：5种手机品牌风格（小米/华为/OPPO/vivo/iPhone）
- ✅ **实时搜索**：快速查找应用
- ✅ **一键启动**：点击直接打开应用

## 🚀 快速开始

### 方式一：下载预构建的 APK（推荐）

1. 访问 [Releases 页面](../../releases)
2. 下载最新的 APK 文件
3. 传到手机安装即可

### 方式二：GitHub Actions 自动构建

1. Fork 本项目
2. 推送代码后自动构建
3. 在 Actions 页面下载 APK
4. 详见 [GitHub Actions 指南](GITHUB_ACTIONS_GUIDE.md)

### 方式三：本地构建

需要安装 Android Studio，详见 [安装指南](INSTALL_GUIDE.md)

## 📋 分类规则

应用会根据包名自动分类：

- **社交**：微信、QQ、微博、抖音、小红书等
- **办公**：钉钉、企业微信、WPS、飞书等
- **娱乐**：爱奇艺、腾讯视频、网易云音乐等
- **工具**：高德地图、百度网盘、相机等
- **购物**：淘宝、京东、拼多多、美团等
- **金融**：支付宝、各大银行App等
- **游戏**：王者荣耀、和平精英等
- **其他**：无法匹配的应用

## 🛠️ 技术栈

- **语言**：Kotlin
- **UI框架**：Material Design 3
- **架构**：MVVM
- **异步处理**：Kotlin Coroutines
- **列表优化**：RecyclerView + DiffUtil

## 🔐 权限说明

需要的权限：
```xml
<uses-permission android:name="android.permission.QUERY_ALL_PACKAGES" />
```

**用途**：读取手机上已安装的应用列表  
**隐私**：仅本地读取，不上传任何数据
